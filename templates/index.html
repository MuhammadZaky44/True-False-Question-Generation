About

This website was created to objectify my final project and to meet the criteria to publish my work to conference. The title of my final project is Automatic True/False Question Generation for Educational Purposes. I was working on in this project to make a true/false question generator using Natural Language Processing (NLP) methods, using Indonesian text as the baseline.

The original paper that I used as the main reference is Zou et.al. (2022). They were using between 2 frameworks, which are Template-based framework and Generative Framework. In my project I only took Template-based Framework as my reference

The scope of problem of my project are:

- The baseline of text and NLP methods I used is in Bahasa Indonesia

- The main subject that I aim for is between elementary student to first year junior high school student
