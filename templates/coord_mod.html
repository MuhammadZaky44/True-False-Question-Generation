Coordination Modification Template

Coordination modification template use POS Tagger model to detect tagged word in the input sentence. If the program found <Noun> - <CC> - <Noun> sequence, the program will remove first Noun and CC tag, so it will be only the second Noun that is written from the original input sentence.

I strongly recommend to manually check if there is any"dan" in your sentence because the program will mainly works if it's detect"dan" in your sentence(s).

If the sequence is not found, then the program will print out'sequence not found' instead for user friendly purposes.
